# byU 포토부스 v1.3.0 — 무인운영 자가복구 적용

이번 버전의 핵심: **진짜로 동작하는 자가복구(Watchdog)** — 무인운영을 위한 안정성.

---

## 🆕 v1.3.0에서 실제로 바뀐 것

### ⚠️ 자가복구 (Watchdog) — 가짜 → 진짜
이전엔 화면(UI)만 있고 엔진이 없었습니다. 이제 **실제로 감지·복구·기록**합니다:

| 상황 | 이전 | 지금 (v1.3.0) |
|---|---|---|
| 화면이 멈춤(응답 없음) | 그냥 멈춤 | **자동 감지 → 새로고침 복구** |
| 프로그램이 죽음(크래시) | 꺼진 채 방치 | **자동으로 창 재생성** |
| 하트비트 끊김(20초) | — | **자동 복구 시도 (최대 N회)** |
| 복구 이력 | 가짜 텍스트 | **진짜 이력 디스크 기록** |
| 카메라/프린터 재연결 | 화면만 | **실제 이력 + 콘솔 알림** |

→ 관리자 → 시스템 → **자동 복구(Watchdog)** 패널에서 실시간 이력 확인.
→ 무인운영 중 프로그램이 멈춰도 **사람 없이 스스로 살아납니다.**

### 🖨 프린터 (v1.2.0~) — 실제 연동
- 프린터 검색 / 설정창 열기 / 테스트 인쇄 (4×6) 실제 동작

## 설치 후 테스트 체크리스트

### 자가복구 테스트 (이번 버전 핵심) ✅
설치 후 실제로 확인해보세요:
1. byU 실행 → 정상 작동 확인
2. **작업관리자(Ctrl+Shift+Esc)** → byU 프로세스 강제 종료
3. → **잠시 후 byU가 자동으로 다시 켜지는지** 확인
4. 관리자 → 시스템 → 자동 복구 패널에 **복구 이력이 진짜로 기록**됐는지 확인

> 참고: 화면 멈춤·크래시 복구는 자동입니다. 완전 종료 후 자동 재실행은 윈도우 시작프로그램 등록 시 가장 안정적입니다(설치 시 바로가기 자동 등록).

### 프린터 테스트
관리자(우상단 1.5초 길게 → PIN `000000`) → 프린터 설정:
- [ ] 프린터 검색 → 목록 표시
- [ ] 프린터 설정창 열기 → 윈도우 인쇄 대화상자
- [ ] 테스트 인쇄 (4×6) → 실제 출력
