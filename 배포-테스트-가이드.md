# byU 포토부스 v1.5.0 — 실제 웹캠 촬영

이번 버전부터 **진짜 웹캠**으로 라이브뷰 + 촬영이 됩니다 (이전엔 시뮬레이션이었음).

## 🆕 바뀐 것
- **실시간 웹캠 라이브뷰** — 준비·촬영 화면에 실제 카메라 화면 표시 (셀피 좌우반전)
- **실제 촬영** — 카운트다운 후 셔터 순간 진짜 프레임 캡처
- **찍은 사진이 인화지에** — 선택·미리보기·인쇄에 실제 촬영 사진 반영
- 카메라 없으면 → "카메라를 찾을 수 없어요" 안내 + 실루엣 폴백

## 설치 후 테스트
1. 새 `byU-PhotoBooth-Setup.exe` 설치 (자동 업데이트로도 받아짐)
2. 촬영 시작 → **준비 화면에서 웹캠 화면이 뜨는지** 확인
   - 처음 한 번 윈도우가 "카메라 허용?" 물으면 **허용**
3. 촬영 → 찍힌 사진이 선택·인화 미리보기에 나오는지 확인

### 카메라가 안 보이면 (Windows)
- **설정 → 개인 정보 → 카메라** → "앱이 카메라에 액세스하도록 허용" + "데스크톱 앱이 카메라에 액세스" **켜기**
- 다른 프로그램(줌·EOS Utility 등)이 카메라를 점유 중이면 종료
- USB 웹캠이 장치관리자에 정상 인식되는지 확인
